prod
====

Output direct product of lines of each files.

Usage
-----

```
$ prod [OPTION]... [FILE]...
Output direct product of lines of each files.

Options:
  -s, --separator=STRING    use STRING to separate columns (default: \t)
      --help                display this help text and exit
      --version             display version information and exit
```
